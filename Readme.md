# PSIQUE-MASAS: Arquitecto de Consenso

Aplicación de análisis de propaganda basada en la psicología de las masas de Gustave Le Bon. Utiliza DeepSeek como motor de IA.

## Despliegue en Vercel

1. Conecta este repositorio a Vercel.
2. Añade la variable de entorno `DEEPSEEK_API_KEY` con tu API key de DeepSeek.
3. Despliega.

## Desarrollo local

```bash
npm install
npm run dev
